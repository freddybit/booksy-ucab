# Booksy-UCAB
Booksy UCAB es una aplicación web desarrollada para facilitar la compra, venta y consulta de libros entre los miembros de la comunidad universitaria de la UCAB. Su propósito es crear un espacio confiable y accesible donde estudiantes, docentes y personal puedan intercambiar libros académicos y de interés general dentro del entorno universitario.
## Tech Stack
- Client: Vue
- Server: .NET
